Bokstavering

I denna kata ska vi öva på testdriven javascriptutveckling med Karma, Jasmine och AngularJS

Målet är att bokstavera inmatningen från ett inmatningsfält med det internationella bokstaveringsalfabetet.

Tänk att du sitter i telefon med någon engelsktalande som aldrig har sett ditt namn, men behöver kommunicera det för att din adress ska bli rätt.

Standardfält för inmatning i html bundet till en variabel via angular-attributet ng-model
<input ng-model="alpha"/>
Html-template har tillgång till variabler definierade av ng-model och kan eka ut dessa med dubbla måsvingar
{{ alpha | spell }}

Meningen är att denna instruktionssida inte ska behöva användas för att bekräfta att implementationen av filtret utförts korrekt. Karma/Jasmine-testerna bör vara tillräckligt.

bravo, echo, romeo, golf, sierra, tango, romeo, oscar, echo, mike

{{ alpha | spell | commaSeparated }}

Kata

  1. Skriv ett kanariefågel-test
  2. Skriv tester i spell.spec.js
  3. Implementera filter i app.js
  4. Kör testerna kontinuerligt med npm test

Som avslutning, starta webbserver med npm start och visa denna sida via http://localhost:8000
